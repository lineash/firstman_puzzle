# firstman_puzzle
객체지향프로그래밍 과제입니다.
----------------------------------------------------------------------
1. 빈칸 주변의 퍼즐조각을 클릭하면 조각이 움직입니다.
2. 총 몇번 움직였는지 카운트하고 게임이 끝나면 알려줍니다.
3. 118번줄의 mixCount를 통해 섞을 횟수를 수정하실 수 있습니다. 초기값은 100 입니다.
